ctrl+alt+t : 터미널

cd : 상위 디렉토리로
cd ~/... : 디렉토리 이동
cd ~ : 홈으로 이동
ls -al : 디렉토리 목록 보기
pwd : 현재 디렉토리 경로 출력
mkdir : 디렉토리 생성
rm : 파일/디렉토리 삭제
which : 위치

cp .. .. : 복사
mv .. .. : 이동/이름변경
cat : 파일 내용 출력
nano : 간단한 편집기
echo "..." : 출력해라
gedit : 텍스트 편집기(GNOME 기반)

top : 실시간 자원 사용
ps : 실행중인 프로세스 보기

sudo apt update : APT 패키지 목록을 갱신
sudo apt upgrade : 설치된 패키지 업그레이드
sudo apt install : 패키지 설치
sudo apt remove : 패키지 삭제
sudo : 관리자 권한

명령어1 ; 명령어2 : 두개 실행
명령어1 && 명령어2 : 앞 명령어가 성공했을때만 실행
명령어1 || 명령어2 : 앞 명령어가 실패했을때만 실행
